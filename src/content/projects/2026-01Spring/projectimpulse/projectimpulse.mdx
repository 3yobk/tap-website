{/* Project Description */}

Project Impulse is an AI-controlled spring-powered jumping robot designed to demonstrate how artificial intelligence, robotics, and mechanical engineering can work together in a real-world system. The robot uses a motor-driven spring compression system to store energy and launch itself vertically into the air. Our project combines electronics, programming, 3D printing, and mechanical design into one interactive robotics platform.

The workshop teaches participants how AI can interact with physical hardware systems using simple electronic circuits and voice-controlled software. Students will learn how circuits work, how robotics systems are controlled, and how AI can communicate with hardware components in real time.

Project Impulse also demonstrates how modern technologies such as Raspberry Pi controllers, motor drivers, servo systems, and 3D printed parts can be combined to build custom robotics projects from scratch.

The Technology Ambassador Program at [GGC](http://www.ggc.edu/tap) strives to break misconceptions about the IT field by providing engaging and hands-on workshops for students of all backgrounds. TAP students create interactive projects that encourage interest in STEM, engineering, robotics, and computer science.

## Steps

1. Build the LED circuit using the breadboard, resistor, jumper wires, and Arduino Uno.

2. Connect the circuit ground to the Arduino GND pin.

3. Test the LED using the 3.3V output pin.

4. Open the Delta AI software folder.

5. Run the `run_windows` startup script.

6. Install Python if prompted by the setup process.

7. Wait for the Delta AI system to initialize.

8. Move the LED power wire from the 3.3V pin to GPIO Pin 11.

9. Use Delta AI commands such as:
   * `Turn light on`
   * `Blink light fast`
   * `Blink light slowly`

10. Observe how the AI system controls the physical electronics in real time.
